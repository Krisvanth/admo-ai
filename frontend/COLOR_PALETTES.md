# Admo AI - Color Palettes

Here are the three color palettes designed for Admo AI. You can switch between them by updating the `tailwind.config.js` file.

## Option 1: "Deep Intelligence" (Navy & Gold)
**Vibe:** Professional, Academic, Premium, Trustworthy.

### Tailwind Config
```javascript
colors: {
  primary: {
    50: '#eff4ff',
    100: '#dbe6fe',
    200: '#bfd3fe',
    300: '#93bbfd',
    400: '#609afa',
    500: '#3b82f6',
    600: '#2563eb',
    700: '#1d4ed8', // Base Primary (Royal Blue)
    800: '#1e40af',
    900: '#1e3a8a', // Deep Navy (Sidebar)
    950: '#172554',
  },
  secondary: {
    50: '#fffbeb',
    100: '#fef3c7',
    200: '#fde68a',
    300: '#fcd34d',
    400: '#fbbf24',
    500: '#f59e0b', // Base Secondary (Amber/Gold)
    600: '#d97706',
    700: '#b45309',
    800: '#92400e',
    900: '#78350f',
    950: '#451a03',
  }
}
```
